# NFC One Latest Photo Site

這版只會顯示 `photos` 資料夾裡最新的一張照片。

## 之後怎麼換照片

1. 打開 GitHub Repository
2. 進入 `photos` 資料夾
3. 點 `Add file` → `Upload files`
4. 上傳新照片
5. 按 `Commit changes`

網站會自動顯示排序最新的一張照片。

## 最乾淨的做法

如果你希望舊照片不要留在 GitHub：

1. 進入 `photos` 資料夾
2. 刪掉舊照片
3. 上傳新照片
4. Commit changes
