# NFC Auto Photo Gallery

這版不需要改程式，也不需要改照片檔名。

## 新增照片
1. 打開 GitHub Repository
2. 進入 `photos` 資料夾
3. 點 `Add file` → `Upload files`
4. 直接拖進照片
5. 按 `Commit changes`

支援 JPG / JPEG / PNG / WEBP / GIF。

## 小建議
如果想控制排序，可以用日期當檔名前綴，例如：

- 2026-07-10-bear.jpg
- 2026-07-11-tea-egg.jpg
